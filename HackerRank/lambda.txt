cube = lambda x: (x*x*x) 

def fibonacci(n):
    ls=[]
    for i in range(n):
        if i==0:
            ls.append(0)
        elif i==1:
            ls.append(1)
        elif i>1:
            res = ls[i-2]+ls[i-1]
            ls.append(res)

    return ls

if __name__ == '__main__':
    n = int(input())
    print(list(map(cube, fibonacci(n))))


Title:Map and Lambda Function
Approach: Generate the first n Fibonacci numbers using iteration and store them in a list, then apply a lambda function to cube each value using map().

Time & Space Complexity: Time complexity is O(n) and space complexity is O(n).
